About VCF Format

                            
VCF (Variant Call Format) is a standard file format for storing gene sequence variations. The generated file includes:

                            
    
                                
  • Header with metadata and format version
    • 
                                
  • Chromosome, position, and rsID for each variant
    • 
                                
  • Reference and alternate alleles
    • 
                                
  • Genotype information
